# WebdriverIO meetup

WebdriverIO meetup examples

## Install WebdriverIO

```
npm install @wdio/cli
```

## Installation

First clone repository

```bash
git clone https://github.com/jerryosorio2/webDriverIOMeetup.git
```

Install packages

```bash
npm install
```

Execute test

```bash
npm run headless

```
